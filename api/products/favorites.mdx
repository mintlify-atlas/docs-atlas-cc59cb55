## Add to Favorites

<Note>
  Requires authentication. Adds a product to the authenticated user's favorites list.
</Note>

### Endpoint

```
POST /products/favourite/{productCode}
```

### Path Parameters

<ParamField path="productCode" type="integer" required>
  The unique product code to add to favorites
</ParamField>

### Response

Returns a success message string.

### Example Request

<CodeGroup>

```bash cURL
curl -X POST "https://api.example.com/products/favourite/10001" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/products/favourite/10001', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const message = await response.text();
```

```python Python
import requests

headers = {'Authorization': 'Bearer YOUR_TOKEN'}

response = requests.post(
    'https://api.example.com/products/favourite/10001',
    headers=headers
)
message = response.text
```

</CodeGroup>

### Example Response

```
El producto con código 10001 se agregó a favoritos correctamente.
```

### Error Messages

```
Usuario no encontrado.
```

```
Producto no encontrado.
```

```
El producto ya está en favoritos.
```

## Get Favorite Products

<Note>
  Requires authentication. Returns the list of favorite products for the authenticated user.
</Note>

### Endpoint

```
GET /products/user/favourite-products
```

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

### Response

Returns an array of favorite product objects.

<ResponseField name="favorites" type="array">
  Array of favorite products
  
  <ResponseField name="id" type="integer">
    Product ID
  </ResponseField>
  
  <ResponseField name="title" type="string">
    Product title
  </ResponseField>
  
  <ResponseField name="description" type="string">
    Product description
  </ResponseField>
  
  <ResponseField name="mediaSrc" type="array">
    Array of image URLs
  </ResponseField>
  
  <ResponseField name="price" type="float">
    Product price
  </ResponseField>
  
  <ResponseField name="stock" type="integer">
    Available stock
  </ResponseField>
  
  <ResponseField name="productCode" type="integer">
    Unique product code
  </ResponseField>
</ResponseField>

### Example Request

<CodeGroup>

```bash cURL
curl -X GET "https://api.example.com/products/user/favourite-products" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/products/user/favourite-products', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const favorites = await response.json();
```

```python Python
import requests

headers = {'Authorization': 'Bearer YOUR_TOKEN'}

response = requests.get(
    'https://api.example.com/products/user/favourite-products',
    headers=headers
)
favorites = response.json()
```

</CodeGroup>

## Notes

- All favorite endpoints require authentication
- Users can add the same product to favorites only once
- Only active products are included in the favorites list
- Removing a favorite product that doesn't exist in favorites will still return success
- Favorite actions emit events to the Communication API (Core) system:
  - `emitAddFavorite` when adding to favorites
  - `emitRemoveFavorite` when removing from favorites
- The `GET /products/user/favourite-products` endpoint extracts the user email from the JWT token in the Authorization header
