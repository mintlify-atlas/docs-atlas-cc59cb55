---
title: "Get Cart"
api: "GET /cart/{id}"
---

Retrieves a specific cart by its ID. Requires user authentication and an active session.

## Authentication

This endpoint requires a valid Bearer token in the Authorization header.

<ParamField path="id" type="integer" required>
  The unique identifier of the cart to retrieve
</ParamField>

## Request Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
  
  Example: `Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...`
</ParamField>

## Response

<ResponseField name="id" type="integer">
  The unique identifier of the cart
</ResponseField>

<ResponseField name="user" type="object">
  The user who owns this cart (reference)
</ResponseField>

<ResponseField name="finalPrice" type="float">
  The total price of all items in the cart
</ResponseField>

<ResponseField name="items" type="array">
  Array of cart items
  
  <ResponseField name="id" type="integer">
    Cart item unique identifier
  </ResponseField>
  
  <ResponseField name="quantity" type="integer">
    Quantity of the product
  </ResponseField>
  
  <ResponseField name="product" type="object">
    Product details
    
    <ResponseField name="id" type="integer">
      Product ID
    </ResponseField>
    
    <ResponseField name="title" type="string">
      Product title
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Product description
    </ResponseField>
    
    <ResponseField name="price" type="float">
      Product price
    </ResponseField>
    
    <ResponseField name="stock" type="integer">
      Available stock
    </ResponseField>
    
    <ResponseField name="productCode" type="integer">
      Unique product code
    </ResponseField>
    
    <ResponseField name="mediaSrc" type="array">
      Array of image URLs for the product
    </ResponseField>
  </ResponseField>
</ResponseField>

## Status Codes

- **200 OK** - Cart retrieved successfully
- **401 Unauthorized** - Invalid or missing authentication token, or user session is not active
- **404 Not Found** - Cart with the specified ID does not exist

<CodeGroup>
```bash cURL
curl -X GET "https://api.example.com/cart/1" \
  -H "Authorization: Bearer YOUR_TOKEN_HERE"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/cart/1', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN_HERE'
  }
});

const cart = await response.json();
console.log(cart);
```

```python Python
import requests

response = requests.get(
    'https://api.example.com/cart/1',
    headers={'Authorization': 'Bearer YOUR_TOKEN_HERE'}
)

cart = response.json()
print(cart)
```
</CodeGroup>

## Notes

- The user must have an active session to access this endpoint
- Only the cart owner can retrieve their cart (validated via JWT token)
- The cart entity uses lazy loading for the user relationship to optimize performance
