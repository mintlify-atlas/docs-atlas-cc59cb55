# Ecommerce Backend API

A production-ready Spring Boot backend service for e-commerce applications, featuring complete product catalog management, user authentication, shopping cart functionality, and event-driven architecture with Kafka.

## Technology Stack

The Ecommerce Backend is built with:

- **Spring Boot 3.5.4** - Modern Java framework
- **Spring Security** - JWT authentication and authorization
- **Spring Data JPA** - Database access with Hibernate
- **MySQL** - Primary database
- **Apache Kafka** - Event streaming and messaging
- **Keycloak** - Identity and access management
- **Docker** - Containerization and deployment
